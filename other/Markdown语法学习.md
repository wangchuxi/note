## 标题的创建
(标题后面必须有一个空格，这是markdown基本语法。以此类推六级标题六个井号；)

``` 
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 列表

```
无序列表 用“*”“-”“+”都可以。
* 1         .1
- 2         .2 
+ 3         .3
```
>

Example:
* 无序列表1
- 无序列表2
+ 无序列表3

```
有序列表(就不应用实例了一眼就明白)
1.          1. 
2.          2.
3.          3.
```
